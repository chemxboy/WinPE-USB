BS Explorer - specially written for use with Windows PE (Windows Preinstallation Environment) the shell. Replaces with itself the command line in standard Windows PE. Incorporates the menu "Start-up", the panel of tasks, area of notices with hours (tray), a desktop and the panel of fast start.

Necessary files and updating can be downloaded freely from here:
www.kursknet.ru/~boa


In more detail read in a file from the complete set of delivery readme.txt
For converting Nu2menu XML of a configuration file in a format ini - files Bs Explorer use the program
Nu2 menu to Bs Explorer convert and manage.
Necessary files and updating can be downloaded freely from here:
www.kursknet.ru/~boa


At use of the program in quality default shells, please comment in all plug-ins similar lines:
[SetupReg. AddReg]
0x1, "Setup", "CmdLine", "nu2menu.exe"
